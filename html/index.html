Project: Dice Game

Instructions: roll the dice three times to get a calculated score. Compete with the dealer's score. The score is calculated as follows:

  • If a 1 is rolled with either dice, the score is 0
  • If a pair is rolled, the score is added up and doubled
  • Otherwise, the two dice are added up
